What is the 'class' attriute in HTML? Class attributes are attributes which are owned by the class itself. They will be shared by all the instances of the class. Therefore they have the same value for every instance. We define class attributes outside all the methods, usually they are placed at the top, right below the class header.
What is the differece between the 'id' attriute and the 'class' attriute of HTML elements? A Class name can be used by multiple HTML elements, while an ID name must only be used by one HTML element within the page.
